Stair Step

A linear function of x that has been quantized to discrete levels using the FLOOR function. The FLOOR function rounds a real number down to the next smallest integer. The width parameter b controls the width of each step, and the amplitude Amplitude, A0 controls the amplitude of the first step.

Parameters:

  • Amplitude, A0
  • Center, x0
  • Width, b
  • Power, n

Support: infinite

Area: zero

Symmetry: odd
