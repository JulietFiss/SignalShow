2D Fourier Basis Function (Complex Sinusoid)

The 2D Fourier Basis Function is a complex exponential with a linear phase that varies in both the x and y directions. This function can be represented as sums of products of sinusoids. The 2D Fourier Basis function has a constant magnitude.

To edit the complex function, edit the parameters of any of the component sinusoids. The parameters of the other sinusoids making up the function will automatically update.

In terms of period:

Parameters:

  • Amplitude, A0
  • Period, X0
  • Initial Phase, ɸ0
  • Center, x0*
*the center parameter has been added to SignalShow for convenience
