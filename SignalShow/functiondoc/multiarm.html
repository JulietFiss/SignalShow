Multiarm with Circular Subapertures

Aperture function for a sparse aperture imaging system with circular subapertures. The multiarm is made up of equally-spaced radial "arms". Each arm is made of equally spaced subapertures.

Parameters:

  • Amplitude, A0
  • X Center, x0
  • Y Center, y0
  • Radius, r1, the radius of each subaperture
  • Spacing, the distance between supaperture centers within each arm
  • Number of Arms
  • Number of Apertures (within each arm)
