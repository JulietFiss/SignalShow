Multiarm with Cassegrain Subapertures

Aperture function for a sparse aperture imaging system with Cassegrain subapertures. The multiarm is made up of equally-spaced radial "arms". Each arm is made of equally spaced subapertures.

Parameters:

  • Amplitude, A0
  • X Center, x0
  • Y Center, y0
  • Inner Radius, r0, the inner radius of each subaperture
  • Outer Radius, r1, the outer radius of each subaperture
  • Spacing, the distance between supaperture centers within each arm
  • Number of Arms
  • Number of Apertures (within each arm)
