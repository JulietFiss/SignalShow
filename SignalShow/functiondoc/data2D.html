2D Data

Load 2D data from a text file or an image.

Text file: Data elements should be saved in a human-readable text file, delimited by white space.

The first element in the file must be the width of the image (integer). The second element in the file must be the height of the image (integer). The rest of the elements are pixels in the image (double precision floating point).

Image: must be a grayscale image.
