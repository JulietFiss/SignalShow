Binomial Noise

Each element in the array is created by summing the results of n Bernoulli trials, where the outcome of each Bernoilli trial is either a 1 (occurring with probability p) or a 0 (occuring with probability 1-p). The binomial distribution can be thought of as the discrete approximation to the Gaussian distribution.

Parameters:

  • n, number of Bernoulli trials
  • p, probability of the outcome "1" occurring in a single Bernoulli trial.
  • seed, the seed used for the pseudorandom number generator.

Mean: np

Variance: np(1-p)
