Threshold

The threshold operator sets any value in the input greater than or equal to the threshold to the maximum value, and any value in the input less than the threshold to the minimum value.

Parameters:

  • Threshold
  • Minimum: any value in the input less than Threshold will be set to Minimum.
  • Maximum: any value in the input greater than or equal to Threshold will be set to Maximum.
